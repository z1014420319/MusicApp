# MusicApp
基本功能：
  1榜单随机歌曲推荐
  2创建和删除播放列表
  3将歌曲添加到或从播放列表中删除
  4单曲，随机，和顺序播放歌曲
  5搜索歌曲
  6云用户密码管理
  7本地用户信息存储
  8已有的用户名qwer密码qwer,用户名asdf密码,asdf,用户名zxcv密码zxcv
